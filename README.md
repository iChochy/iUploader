**软件介绍：**

​	`iUpload`主要功能将图片上传至七牛云，返回 `Markdown` 格式的链接到剪贴板



**功能介绍：**

1. 图片本地压缩 

2. 图片右键上传

3. 图片复制上传

4. 图片拖拽上传

5. https加密上传

   

**开发：**

​	继承七牛云SDK，使用`swift`开发，App自签上传凭证，自动选择存储区域，通过https加密上传。
